# project_2

這是一份運用vue3和tailwind技術的專案，實作一個義大利麵餐廳的官方網頁。目前仍在進一步完善中，目標是完成各個頁面，並串接簡單的web API。

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
